git init------------> This command is used to create a new Git repository. It initializes an empty Git repository in the current directory or in the specified directory.

git add------------->This command is used to add changes to the staging area in preparation for committing. It can be used to add specific files or directories, or to add all changes in the working directory.

git commit----------> This command is used to create a new commit with the changes that have been added to the staging area. A commit is a snapshot of the repository at a specific point in time, and it includes a commit message describing the changes.

git push------------> This command is used to upload local repository changes to a remote repository. It sends the committed changes from the local repository to the remote repository.

git pull------------> This command is used to download changes from a remote repository and merge them into the local repository. It retrieves changes from the remote repository and integrates them with the local repository.
